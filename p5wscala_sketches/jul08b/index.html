Keyboard controls

WASD to move (hold shift to move faster)
'q'/'e' to change the terrain generator
'1'/'3' to change the combiner function (only affects generators 4-7)
'z'/'x' to make the grid bigger/smaller

Mouse controls

Button2 -> Zoom
Button3 -> Translate frame
Shift+Button3 -> Screen rotate
Button1 -> Rotate frame
Shift+Button1 -> Zoom on region
Button1 + 2 clicks -> Align camera with world
