This sketch generates a random mathematical AST composed of various unary and binary functions, constants, and the variables x/y.

Keyboard controls

'q' to switch between viewing the function ASTand viewing the function plot
space to generate a new random function
wasd/arrow keys to move around
-/= to zoom

Mouse controls

scroll wheel to zoom
Move the mouse from left to right to change the plot's brightness

Source code:Dec02b
